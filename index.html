AUTOBOXING/UNBOXING

AUTOBOXING: Autoboxing in Java is the automatic conversion of a primitive data type (e.g., int, double) into its corresponding wrapper class object (e.g., Integer, Double) by the Java compiler.

UNBOXING: Unboxing in Java is the automatic conversion of a wrapper class object (e.g., Integer, Double) into its corresponding primitive data type (e.g., int, double) by the Java compiler.

Example: Display the 3 days of movie collection using Autoboxing

public class Main {
    public static void main(String[] args) {
        // Autoboxing: Converting primitive int to Integer
        Integer day1Collection = 100; // Day-1 collection in crores
        Integer day2Collection = 120; // Day-2 collection in crores
        Integer day3Collection = 150; // Day-3 collection in crores

        // Unboxing: Converting Integer to primitive int
        int totalCollection = day1Collection + day2Collection + day3Collection;

        // Display the collections
        System.out.println("Day-1 Collection: " + day1Collection + " Crores");
        System.out.println("Day-2 Collection: " + day2Collection + " Crores");
        System.out.println("Day-3 Collection: " + day3Collection + " Crores");
        System.out.println("Total Collection: " + totalCollection + " Crores");
    }
}

AUTOBOXING METHODS

Here are the key methods related to autoboxing in Java:

1. Autoboxing (Primitive to Wrapper):

This is the automatic conversion of a primitive data type to its corresponding wrapper class. For example:

int a = 10;
Integer obj = a;  // Autoboxing: int to Integer

2.Unboxing (Wrapper to Primitive):

This is the automatic conversion of a wrapper class object back to its corresponding primitive type. For example:

Integer obj = 10;
int a = obj;  // Unboxing: Integer to int

3.Wrapper Class Methods

  • valueOf():

    • Converts a string or primitive value to the corresponding wrapper class object.

            Integer obj = Integer.valueOf(10); // converts int to Integer
  • parseInt():

    • Converts a string to a primitive int.

            int num = Integer.parseInt("123");  // converts string to int
  • intValue():

    • Converts an Integer object to a primitive int.

            int num = obj.intValue();  // Integer to int

AUTOBOXING/UNBOXING boolean and chatacter value

Autoboxing and unboxing in Java are mechanisms that allow automatic conversion between primitive types and their corresponding wrapper classes. This is particularly useful when working with collections like ArrayList, which can only store objects and not primitive types.

Hereâ€™s how it works for boolean and char:

1.Autoboxing:

Automatic conversion of a primitive type to its corresponding wrapper class object.

Example for boolean:

boolean boolValue = true;
Boolean boolObject = boolValue; // Autoboxing

Example for char:

boolean boolValue = true;
Boolean boolObject = boolValue; // Autoboxing


2.Unboxing:

Automatic conversion of a wrapper class object back to its corresponding primitive type.

Example for boolean:

Boolean boolObject = Boolean.TRUE;
boolean boolValue = boolObject; // Unboxing

Example for character:

Character charObject = 'B';
char charValue = charObject; // Unboxing


Practical example Combining both

import java.util.ArrayList;

public class Main {
    public static void main(String[] args) {
        // Autoboxing
        ArrayList charList = new ArrayList<>();
        charList.add('X'); // Autoboxing: 'X' -> Character

        // Unboxing
        char ch = charList.get(0); // Unboxing: Character -> 'X'

        System.out.println("Character in List: " + ch);

        // Autoboxing with boolean
        ArrayList boolList = new ArrayList<>();
        boolList.add(true); // Autoboxing: true -> Boolean

        // Unboxing
        boolean bool = boolList.get(0); // Unboxing: Boolean -> true

        System.out.println("Boolean in List: " + bool);
    }
}
